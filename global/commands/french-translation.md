# French Translation Skill

Use when editing French translation files (*.json with French content, fr.json, etc.)

## Accent Rules - NEVER Omit

| Letter | Examples | Common in |
|--------|----------|-----------|
| é | ajouté, activé, désactivé, créé, difficulté | Past participles, nouns |
| è | règle, très, après | Before silent consonants |
| ê | être, même, fenêtre | Replacing old 's' |
| à | à, déjà, voilà | Preposition "at/to" |
| ù | où | "where" |
| ç | français, reçu, façon | Before a, o, u |
| î, ï | naïf, maïs | Rare |
| ô | côté, hôte | Rare |

## Common Mistakes to Avoid

```
WRONG → CORRECT
ajoute → ajouté (past participle)
desactive → désactive
difficulte → difficulté
munitions illimitees → munitions illimitées
armes aleatoires → armes aléatoires
vires → virés
```

## Gaming Terms - Keep in English

These stay English even in French translations:
- headshot, kill, spawn, respawn
- CT, T (team names)
- bot, mod, plugin
- FFA, competitive, casual
- Workshop (Steam Workshop)

## Gaming Terms - Translate

```
English → French
tip → astuce
weapons → armes
ammo/ammunition → munitions
map → map (or carte)
difficulty → difficulté
commands → commandes
available → disponible(s)
```

## Sentence Structure

- Use informal "tu" form for player-facing messages (not "vous")
- Keep messages concise like English version
- Example: "Type !help" → "Tape !help" (not "Tapez !help")

## Before Saving

Always verify:
1. All é, è, ê accents present
2. All à accents present
3. ç used before a, o, u when needed
4. Past participles end in -é not -e (ajouté, activé, désactivé)
